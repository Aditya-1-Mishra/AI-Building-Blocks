# AI-Building-Blocks

## What is this repository?

A collection of machine learning, deep learning, natural language processing, computer vision, and generative AI implementations created for learning and experimentation.

## Why does it exist?

This repository documents my journey of understanding AI systems by building and exploring individual concepts, models, and techniques rather than only using complete end-to-end solutions.

## What kinds of projects are inside?

The repository includes implementations and experiments related to:

* Machine Learning algorithms
* Deep Learning models
* Natural Language Processing (NLP)
* Computer Vision
* Generative AI
* AI building blocks and supporting components
